# 📁 Dossier Images — Égéries DSR

Ce dossier contient les images des égéries DSR : Dysa, Seris et Lyria.

## Structure prévue

```
images/
├── image-1.jpg   ← Égérie 1
├── image-2.jpg   ← Égérie 2
├── image-3.jpg   ← Égérie 3
└── image-4.jpg   ← Égérie 4
```

## Comment uploader une image via GitHub

1. Aller dans ce dossier sur GitHub
2. Cliquer sur **"Add file"** → **"Upload files"**
3. Glisser-déposer tes images
4. Cliquer sur **"Commit changes"**

## Format recommandé
- Format : `.jpg`, `.png` ou `.webp`
- Nommage : `dysa.jpg`, `seris.jpg`, `lyria.jpg`, etc.
