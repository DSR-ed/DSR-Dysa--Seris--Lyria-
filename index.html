Prompts image / vidéo / voix

Structures prêtes à copier et réutiliser dans tes workflows créatifs DSR.

Structure prompt image

1. Identité du personnage + lien DSR 2. Posture / expression / tenue 3. Palette et lumière (16:9) 4. Type : photo éditoriale, portrait premium 5. Logo DSR intégré discrètement, non modifié 6. Contraintes : pas cheap, pas cartoon, pas surcharge sci-fi

Structure prompt vidéo

1. Plan de 8 secondes max 2. Action unique claire 3. Mouvement de caméra simple 4. Présentation : Nom, entreprise DSR, rôle 5. Logo DSR visible discrètement dans le cadre 6. Sortie premium : pub, page d'accueil, réseaux

Structure prompt voix

1. Type de voix 2. Tempo 3. Température émotionnelle 4. Niveau de sourire ou gravité 5. Intention business 6. Interdits : caricature, ton robotique, surjeu pub

Implémentation V3

Ce qui a été corrigé et amélioré par rapport à la V2.

Import local 16:9 + cache persistant

Chaque personnage dispose d'un input type=file et d'un bouton bien aligné. L'image lue via FileReader est convertie en Data URL puis stockée dans IndexedDB.

Aspect-ratio : 16/9 forcé sur chaque zone. Cache : IndexedDB — persiste même après fermeture du navigateur. Effacement : bouton "Vider" par personnage.

Export PDF imprimable

Le bouton "Export PDF" déclenche window.print(). La feuille @media print masque la sidebar, les contrôles et force l'affichage de tous les onglets en continu.

Fond blanc, texte sombre, ombres supprimées, navigation masquée.

Navigation latérale + modes

Navigation fixe à gauche sur desktop, empilée sur mobile. Mode client masque les blocs .internal-only, mode interne masque les blocs .client-only. Un seul fichier HTML.

Ordre de travail recommandé

1. Valider skills (ok V3) 2. Importer planches officielles → cache 3. Ajuster prompts image selon rendu réel 4. Ajuster prompts vidéo et voix 5. V4 : prompts photo réseaux + vidéo DSR séquencée
